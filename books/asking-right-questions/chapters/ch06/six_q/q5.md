# Q5 · 与其他章节的关系

## (a) 本章承接了前面哪些章节的铺垫

**承接 ch02(第1章 学会提出好问题)**:
- ch02 提出了**价值观决定人与人之间的互动**并列出四大核心价值观(自主性、好奇心、谦恭有礼、以理服人者逢之必敬)——那时价值观还只是批判性思考者的**主体**属性(你应该持有的)。ch06 把价值观**客体化**:价值观不只是自己的信念,也是别人隐藏在论证中的假设,需要被识别、命名、评价。这是 ch02 "价值观"概念的一次**认识论翻转**。
- ch02 区分的**强势批判性思维** vs **弱势批判性思维**在 ch06 获得操作化路径:只有把自己和对方的**隐含假设**同时摊开,才可能真正"一视同仁地质疑一切主张"。弱势批判性思维的本质往往就是只质疑对方的假设、不质疑自己的假设。
- ch02 强调的**施惠氛围与谦恭有礼**在 ch06 再次被要求:当识别到作者持有的价值观假设时,态度是追问理由而非嘲讽立场。

**承接 ch03(第2章 论题和结论是什么)**:
- ch03 建立的**描述性论题 vs 规定性论题**二分在 ch06 被延续为**描述性假设 vs 价值观假设**二分:规定性论题的评价主要依赖价值观假设,描述性论题的评价主要依赖描述性假设。
- 作者特别指出"你应该限制自己寻找假设的范围,只在已经学会怎样去寻找的论证结构(理由+结论)中去寻找"——这是对 ch03 "论题+结论"定位工作的再一次调用。没有 ch03 的结构定位,就没有 ch06 的假设范围锁定。

**承接 ch04(第3章 理由是什么)**:
- ch04 的"理由是模具,结论据此成型"在 ch06 被延伸:**模具本身有看不见的支架**——价值观假设让模具能够**倾向于**特定结论,描述性假设让模具本身**站得住**。
- ch04 的**施惠原则**在 ch06 获得更深层应用:要施惠,就不能只评价作者说出来的理由,还要替作者补全他没说出来的假设再评价——这是"先假设论证最无懈可击"的另一种实现路径。
- ch04 的"先有理由再有结论"在 ch06 被扩展为"先有**假设 + 理由**才能有结论"——这条完整推理链如果被颠倒(逆向逻辑),假设就会跟着结论一起被修剪,成为作者事后合理化的道具。
- ch06 raw.txt 中明确引用 ch04 的核心话语:"涉及规定性论题的完整推理离不开理由和价值观假设"——这一表述把价值观假设正式接入 ch04 的"论证 = 结论 + 理由"模型。

**承接 ch05(第4章 哪些词语意思不明确)**:
- ch05 处理**显性词义**的歧义,ch06 通过**定义性假设**(definitional assumption)把歧义问题重新接回假设层——作者在本章汽车案例中明确说明:当推荐者所谓的"驾驶"含义与新任务含义不同,这就是一条定义性假设。定义性假设是 ch05 歧义识别的直接产物,被本章纳入描述性假设的一个子类。
- ch05 关于**附加感情色彩的词**的警告在 ch06 获得更深一层的解释:这类词之所以"让思维短路",正是因为它们偷偷携带了价值观假设——它们用情感包装绕过了读者的假设审查通道。

## (b) 本章为后面哪些章节做准备

**直接后续(评价工具的深化)**:
- **ch07《推理过程中有没有谬误》**:谬误可分为两大类——形式谬误(逻辑结构错误)与非形式谬误(多数涉及假设)。本章建立的"假设识别"工具是 ch07 识别那些涉及隐含假设的谬误(如窃取论题、滑坡谬误、虚假两难、稻草人、诉诸权威)的前置技能。没有 ch06 的假设审查,很多谬误在表面看起来完全合乎逻辑。
- **ch08《证据的效力(一):直觉、个人经历、典型案例、当事人证词和专家意见》**:每一种证据类型都携带自己的描述性假设——"个人经历能代表普遍""典型案例有推断力""专家意见可靠""当事人证词真实"。ch08 的证据效力评价本质上就是对这些假设的专项审查。
- **ch09《证据的效力(二):个人观察、研究报告和类比》**:研究报告的可信度依赖一整组描述性假设(样本代表性、测量有效性、操作定义合理性、因果识别策略)。ch09 是 ch06 描述性假设在科学研究场景下的专业版。类比论证则依赖"相似性足够大"这一关键假设,本章已为之打好基础。

**对因果与定量证据的铺垫**:
- **ch10《有没有替代原因》**:因果归因的关键是识别"除了作者宣称的原因,是否还有其他原因能解释相同现象"——这本质上是在审查作者的描述性假设(因果独立性、混杂控制、时间序列解释)。
- **ch11《数据有没有欺骗性》**:统计数据的隐含假设(样本选择、测量工具、定义边界、呈现方式)在本章已经被提及(储备银行"相貌"研究),ch11 会将其扩展为一整套专项工具。

**对结构型审查的铺垫**:
- **ch12《有什么重要信息被省略了》**:省略信息与隐含假设在概念上是一枚硬币的两面——被省略的信息往往是作者赖以成立但不敢或不愿明说的假设。ch06 的假设识别为 ch12 的"省略识别"提供了第一把手电筒。
- **ch13《能得出哪些合理的结论》**:同一组理由可支撑多个合理结论——核心原因正是**不同的价值观假设** + **不同的描述性假设**会导向不同的连接路径。ch13 的多元结论生成机制,在本章已被奠基。

## 本章在全书坐标系中的角色

若 ch02 是宪法、ch03 是地基、ch04 是承重梁、ch05 是梁上砖的标签清理,那么 **ch06 就是让整个论证大厦从"可视"走向"透视"的第一次 X 光检查**——它迫使读者看见结构背后的隐形支架。在此之前的工具都针对显性的论证元素(论题、结论、理由、词义),在此之后 ch07–ch13 的所有评价工具都要穿过假设层——几乎每一种谬误、每一种证据失效、每一种因果误判、每一种数据欺骗、每一种省略策略,都有其赖以运作的假设结构。因此 ch06 可以视为全书**认识论升级**的关键章:读者从"读论证"升级到"读假设"。

从全书架构看,ch06 也是把 ch02 的**价值观**主题(主体层)与 ch04 的**理由**主题(结构层)接合起来的章节——价值观被降下来进入论证结构内部,理由被升上去承载价值观与世界观。这一双向接合,为后续所有评价工具提供了共同的深层基础。
